MAPFORGE — map & overlay builder

Drop in a background image (a hand-drawn map, a photo, anything), then overlay whatever you like: outline regions (box or freehand), place nodes for cities and join them with roads, rails, dirt, brick, stairs or ladders, scatter image stickers and text labels, drop in an avatar and tap nodes to walk it round the map — then export clean JSON (or a paste-in JS block) for your game, or save the whole board as a PNG. Everything autosaves in this browser. ＋ New map adds another tab.
Board image Type a path relative to this file and hit Import, or Choose file… to embed a picture straight into the map (it's downscaled and saved with everything else).

Map

Paths drawn is the surface every new link and the generator use — tap an existing link to restyle just that one. Floors are the rungs of the map; nodes keep their spots and re-attach to the nearest lane when it changes.

Generator

used by the Emoji slider above
Walks Paths random routes up the floors (paths can share nodes but never cross, same as Slay the Spire), then deals out node types by the sliders. Links come out in the map's Paths drawn surface. Generating replaces the current nodes — Undo gets the old map back.

Grid lattice

Lays out a columns × rows lattice — e.g. 6×6 — and joins every node to the ones next to it (up/down/left/right, plus the corners if diagonals is on). Column w and Row h stretch the spacing by resizing the board; the lattice re-flows live. Building replaces the current nodes — Undo gets them back.

Avatar

In ▶ Play test, tap any node and the avatar walks the paths to it — forwards along the arrows if it can, backtracking if it must. In Select mode, pick a node and hit Send avatar here. The avatar and its position are saved with the map.

Selected node

Tap a node on the board to edit it here. While it's selected, drag it with one finger (the board won't scroll), or nudge it with the arrow keys. Tap a link to select the link instead.

Overlays

▨ Region mode outlines areas — drag a box or sketch a freehand loop (it closes itself when you lift). Ａ Text mode drops a label where you tap. Stickers land in the middle of the board — drag them into place. Tap any overlay in Select mode to edit it in the card alongside; drag the corner handle of a sticker to resize it, drag a region's dots to reshape it.

Selected overlay

Tap a region, sticker or text label on the board (in Select mode) to edit it here.

Export & import

JSON gives you {nodes, edges, regions, stickers, labels, image, …} — ids are stable strings, edges always point start → boss, and the surface is only written when it isn't the plain trail. An embedded board picture is included too, so the file is a complete backup. The JS block is the same data as a paste-in const. Import accepts a single map or a whole {maps:{…}} bundle; the export box is editable, so paste JSON in and hit Import from box.
